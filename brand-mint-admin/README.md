# Brand Mint — Admin Folder

The CEO's operating folder. Everything needed to run the studio from Monday morning is here.

```

├── 04-SALES-PLAYBOOK.md            # Funnel, scripts, qualification rubric
├── 05-OPERATIONS-MANUAL.md         # Inquiry → invoice workflow

├── 08-AI-TOOLSTACK.md              # Google AI Studio + Anthropic stack

```

## Reading order

If you have **30 minutes**: read `00-EXECUTIVE-SUMMARY.md` and `02-BRAND-STRATEGY.md`.

If you have **2 hours**: add `03-SERVICE-CATALOG.md`, `04-SALES-PLAYBOOK.md`, `06-FINANCIAL-MODEL.md`.

Before **first client engagement**: read everything in `ops/` and `contracts/`.

## Source

This folder was scaffolded from the Google AI Studio Design Report (Kimi K2, May 2026) but rewritten to match what Brand Mint *actually* shipped — Plus Jakarta Sans, `#10B981` mint, "We mint brands that compound." tagline, INR pricing, Hyderabad studio. Aspirational suggestions from the source report (Playfair Display, "Where Brands Take Root") have been replaced with the production identity. The structural rigor of the source — voice rubrics, model/temperature settings, 48-hour sprint, quality checklist — is preserved.
