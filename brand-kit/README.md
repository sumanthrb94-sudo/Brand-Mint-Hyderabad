## Quick reference

- **Primary colour:** Mint 3 `#10B981`
- **On-brand background:** Paper cream `#F5F1EA`
- **Display type:** Plus Jakarta Sans (600 for headlines, 500 for editorial)
- **Body type:** Inter
- **Numerals / IDs:** JetBrains Mono
- **Voice:** confident, specific, founder-to-founder. No agency jargon.

## Using these files

- SVGs are master files — scale infinitely, edit in any vector tool
- For raster needs, render at 2× the target size
- Always pair the wordmark with at least 16px of clear space on every side
- Never recolour the gradient mark — use the mono variant if the surface fights it
