BrandMint Studios · 30 Ready-to-Post Instagram Posts
=====================================================

Each post-NN/ folder contains:
  cover.png        Single hook visual (1080x1350). For single-static posts,
                   upload this directly. For Reels, use as the cover/thumb.
                   For Carousels, this is duplicated as slide-01.png.
  slide-01.png     Carousels only — copy of cover.png so slides 1..N are
                   contiguous in upload order.
  slide-02.png     Carousels — body slide 2
  slide-03.png     Carousels — body slide 3
  ...
  slide-NN.png     Carousels — CTA closer (always the LAST slide)
  caption.txt      Copy-paste-ready caption + breathing dots + hashtags
  meta.txt         Pillar, format, recommended post time

How to post:
  Single static     → upload cover.png, paste caption.txt
  Carousel          → upload slide-01.png ... slide-NN.png in order,
                      paste caption.txt
  Reel              → render the matching MP4 from brand-kit/video/...,
                      use cover.png as the Reel thumbnail, paste caption.txt
  Story-poll        → use cover.png as the Story background, add the
                      poll/question sticker manually inside Instagram
