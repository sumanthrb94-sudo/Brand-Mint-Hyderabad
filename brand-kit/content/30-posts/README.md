# @brandmint.studios · 30-Day Instagram Calendar

This folder contains a 30-day Instagram content plan for @brandmint.studios,
split for easy hand-off to designers / schedulers.

## Files

- **00-MASTER-PLAN.md** — voice rules, posting cadence, and the master table
  showing all 30 posts (date, pillar, format, hook) at a glance.
- **post-01-*.md → post-30-*.md** — one file per post, each containing:
    - Pillar + format
    - Hook (first 1-3 seconds)
    - Full caption (≤ 2200 chars, ready to paste)
    - Visual concept (designable in the v13 frame system)
    - Hashtag set (≤ 30, optimized for Hyderabad real-estate discovery)
    - Best time to post + rationale
- **99-CADENCE-AND-METRICS.md** — posting rhythm + metrics-to-watch for
  iteration.

## How to use

1. Open `00-MASTER-PLAN.md` to confirm the 30-day cadence and voice rules.
2. For each scheduled day, open the matching `post-NN-*.md` and hand the
   visual concept to design, the caption to your scheduler.
3. After posting, refer to `99-CADENCE-AND-METRICS.md` to evaluate which
   pillars are winning and adjust week 5+.

All claims are based on public-record information — no invented numbers
or unverifiable awards. Visual concepts are designable in the v13 editorial
frame system (warm-black ground, paper-cream display text, mint accent,
slim letterbox bars, big serif numerals).
